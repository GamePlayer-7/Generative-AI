* Object oriented programming uses objects to design applications and computer programming.
* A Class is a blueprint for creating objects having seperate attributes and methods.
* Polymorphism is done through method overriding and interfaces.
* Method overriding allows child class provide a specific implementation of a method that is already defined in its parent class.
* Encapculation is a concept of wrapping data (variables) and method (functions) together as a single unit. (PUBLIC, PROTECTED, PRIVATE Variables)
   * Public Variable can be accessed from anywhere.
   * Protected variable is written with one underscore _ before the variable name.
   * Private variable is written with double underscore __ before the variable name.
